# Store Depot GitHub Pages Static Build

This folder contains a fully static website package for **Store Depot**, ready to be uploaded to your public GitHub repository to activate GitHub Pages.

## 🚀 Instant Deployment Guide:

1. **Create Repository**: Go to [github.com/new](https://github.com/new) and create a repository (e.g., `storage-depot`).
2. **Make it Public**: Host websites for free on GitHub by setting visibility to **Public**.
3. **Upload files**: Unzip this archive and drag and drop **all files** into your GitHub repository upload box. Alternatively, commit and push normally via Git.
4. **Enable Pages**: Inside the repository page, navigate to **Settings** > **Pages**.
   - Set Source to "Deploy from a branch"
   - Select `main` (or `master`) as the Branch and `/` (root) directory
   - Press **Save**!

Your landing catalog website will be live in 1-2 minutes at:
`https://<your-username>.github.io/<your-repo-name>/`
